What is ARCHived?

ARCHived is a directory of open source GitHub repositories that have been archived by their owners and have seen no activity for at least 12 months. These projects may still be useful, but they no longer have active maintainers.

If you discover a project that fits your needs, you can fork it, adopt it, or reach out to the original authors about transferring ownership — helping bring it back to life.

Collection Criteria

A repository is listed in ARCHived when it meets all of the following:

📦
Archived
Marked as archived by the owner on GitHub
💤
Inactive ≥ 12 months
No commits pushed in the last 12 months
⭐
10+ Stars
At least 10 GitHub stars to filter noise

How It Works

Data is collected daily using the GitHub Search API. To overcome the API's built-in 1,000-result cap, queries are split into multiple star-range buckets (e.g. 10–99 ⭐, 100–499 ⭐, 500–999 ⭐, …) and the results are merged and deduplicated. This allows the dataset to grow well beyond 1,000 projects.

  • Queries run daily via GitHub Actions (00:00 UTC)
  • Results are saved as a static JSON file and served via GitHub Pages
  • Client-side filtering, sorting, and fuzzy search powered by Fuse.js
  • No server-side processing — fully static and fast

Contributing

Found a dead project that isn't listed? Know of a project that has been revived?

  • Open an issue to suggest additions or report inaccuracies
  • Fork the repository and submit a pull request
